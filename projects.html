Catching Ramen Java Game

Download Game

Naruto Falling Ramen is a simple Java-based desktop game where the player controls Naruto as he collects falling ramen to score points. The game runs on a 15-second timer, during which the player must move Naruto using arrow keys or by dragging the mouse to catch as many ramen bowls as possible. After time runs out, the player can choose to restart or exit. The game features basic collision detection, score tracking, and movement constraints within the game window.
